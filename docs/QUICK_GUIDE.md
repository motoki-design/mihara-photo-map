**三原駅周辺現地踏査・景観シミュレーションマップ**

4/27現地踏査の写真117枚と、景観シミュレーション結果を、地図上で見られるWebアプリです。

## ① まずは開いてみる

スマホ・PCどちらでもOK。ブラウザで以下を開くだけ：

ログイン不要。地図と117個のピンが自動で表示されます。

---

## ② 基本の使い方

### 写真を見る
1. 地図上の **ピン** をクリック（タップ）
2. ポップアップが開いて、写真と業務メモが表示される
3. **写真をクリック** すると大きく拡大表示

### カテゴリで絞り込む
画面上の **カテゴリチップ**（[照明][舗装][サイン]…）のチェックを外すと、そのカテゴリ以外のピンが消えます。

例：
- **「サイン」だけONにする** → サインに関係する写真だけ見える
- **「史跡」だけONにする** → 中門跡・天主台跡など史跡関連の写真だけ

### 撮影順を線でたどる
画面上の **☑ 踏査ルート線** で、駅 → 中央ロード → 内港 → 旧市街 → マリンロード → 中門跡 → 城跡 と歩いた順番が破線で表示されます。

### 検索する
画面左の **🔍 検索ボックス** に文字を入れると、ファイル名・場所・キーワードで絞り込めます。

例：「中門跡」「桜山」「街路灯」など

---

## ③ 景観シミュレーション（注目機能）

ピンが **オレンジ色の縁取り** になっている写真には、生成AIで作った景観シミュレーション画像が紐付いています。

### 試し方
1. 駅前広場のオレンジ縁取りピン（IMG_0009）をクリック
2. ポップアップ最下部の **▶ 景観シミュレーション 3案** にボタンが3つ
3. ボタンをクリック → 拡大画面が開く
4. 上部のタブで **元写真／A／B／C** を切り替えて比較

A：和洋中間モダン　／　B：和風行灯　／　C：街路灯撤去・水盤化

---

## ④ ピンの色の意味

| 色 | 意味 |
|-----|------|
| 🟢 緑 | 場所が辞書で確定 |
| 🟠 橙（中） | 他の写真の位置を継承 |
| 🔴 赤 | 撮影時刻から推定 |
| 🟧 縁取り橙 | **シミュレーション画像つき** |

## ⑥ 困ったとき

- **画面が変わらない／古いまま** → **Ctrl + F5**（PC）でハードリロード、もしくはシークレットウィンドウで開く
- **写真が出ない** → ネット接続を確認。少し待って再読込
- **ピンの位置がずれてる** → 座標は守山の概算値。今後校正します
- **詳しいことを知りたい** → 同じフォルダの `USER_MANUAL.md` または `SPEC.md` をご覧ください
