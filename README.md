# MiharaScape

**三原駅周辺現地踏査・景観シミュレーションマップ**

三原市・三原駅周辺公共空間（道路等）デザイン検討業務（#31680）における、現地踏査写真と景観シミュレーション結果を地図上で閲覧できるWebアプリケーションです。

🌐 **公開URL**: https://motoki-design.github.io/mihara-photo-map/

## できること

- 117枚の現地踏査写真（2026-04-27 青木氏撮影）を地図上のピンで一覧
- ピンをクリックで写真サムネ＋場所＋業務関連メモ＋シミュレーション案を表示
- サムネをクリックで原寸（1280px）モーダル拡大
- 9カテゴリ（照明／舗装／サイン／植栽／SF／史跡／景観阻害物／建築／広場）でフィルタ
- 撮影時刻順の踏査ルート線を地図上に表示
- 景観シミュレーション画像（IMG_0009 駅前広場：A/B/C 3案）をタブ切替で比較
- 左サイドバーに117枚の一覧（検索可）
- レスポンシブ対応（PC・スマホ両対応）

## ローカル起動

```bash
cd mihara-photo-map
python -m http.server 8000
```

ブラウザで http://localhost:8000/ を開く。

## ディレクトリ構成

```
mihara-photo-map/
├ index.html                    Leaflet地図UI（単一HTML）
├ README.md                     このファイル
├ docs/                         仕様書・設計書・マニュアル
├ assets/
│  ├ photos.geojson             写真ピンのGeoJSON
│  ├ photos/                    Web用JPG (長辺1280px)
│  ├ thumbs/                    サムネ (長辺200px)
│  └ simulations/               景観シミュレーション画像
├ data/
│  ├ location_dict.json         場所推定→緯度経度の辞書
│  └ simulations.json           写真IDごとのシミュレーションバリアント定義
└ scripts/
   ├ resize_photos.py           原寸→Web版＋サムネ生成
   └ build_geojson.py           ログ＋辞書から GeoJSON 生成
```
